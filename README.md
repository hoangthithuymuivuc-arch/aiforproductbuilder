# AI FOR PRODUCT BUILDERS - Landing Page

Landing page giới thiệu khóa học **Product Academy** - "AI FOR PRODUCT BUILDERS"

## Giới thiệu

Khóa học giúp bạn hiểu tư duy Product cốt lõi và biết cách dùng AI để đi từ Ý tưởng → Phạm vi MVP → Prototype/MVP cơ bản chỉ trong 3 tuần.

## Tính năng

- **Thiết kế hiện đại** - Giao diện dark theme với hiệu ứng neon
- **Responsive** - Tương thích mọi thiết bị
- **Tailwind CSS** - Sử dụng Tailwind CSS qua CDN
- **Google Fonts** - Font Montserrat

## Công nghệ sử dụng

- HTML5
- Tailwind CSS (CDN)
- Google Fonts - Montserrat
- Heroicons (SVG)

## Xem trước

Mở file `index.html` trong trình duyệt để xem landing page.

## Deploy lên GitHub Pages

1. Push code lên repository GitHub
2. Vào **Settings** → **Pages**
3. Chọn branch `main` và folder `/ (root)`
4. Save và chờ deployment

## Cấu trúc thư mục

```
├── index.html          # Landing page chính
├── Demo.html           # File demo (trùng index.html)
├── README.md           # File này
└── .gitignore          # Git ignore file
```
